# Tipico Design System

The shared component library for all Tipico frontend platforms — Games, Sports, Retail, and Account.

## What's inside

This Storybook documents every component in `@tipico/ui`:

- **Live examples** — interactive component playgrounds
- **Usage guidelines** — when to use, when not to use
- **API reference** — all props with types and defaults
- **Best practices** — dos, don'ts, accessibility notes
- **Code snippets** — copy-paste ready

## Getting started

```tsx
import { Button } from "@tipico/ui";

<Button id="cta" variant="primary" size="md">
  Place Bet
</Button>
```
