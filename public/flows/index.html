Flows

AI-powered workflows you can trigger from the Kiro chat. Each flow connects multiple tools into a single automated process.

cx-rebuild Figma → DS Components

Rebuild a selected Figma frame using proper Design System components and tokens. Automatic variant matching and token binding.

Figma Design System
cx-copy Text → Style Guide → Ditto

Extract all text from a Figma frame, audit against the Ditto style guide, produce English translations, optionally push to Ditto with DE/TR variants.

Figma Ditto
cx-audit DS Adoption Check

Scan a Figma frame for Design System adoption gaps, auto-layout issues, and hardcoded values. Produces a prioritized report with fix suggestions.

Figma
cx-project Figma ↔ Jira Sync

Connect a Figma design file to a Jira ticket. Manage phase transitions (Exploration → Review → Build-Ready), update covers and ticket descriptions automatically.

Figma Jira
cx-setup Onboarding

Interactive onboarding flow that walks you through connecting all MCP tools — Figma, Ditto, Atlassian, Recraft, and Bitbucket. Token management and credential setup.

All MCPs

MCP Tools Used

Each flow uses one or more MCP (Model Context Protocol) servers to connect to external tools:

Figma Console

Read/modify designs, inspect tokens, screenshots, execute plugin code

figma-console-mcp

Atlassian Cloud

Jira tickets, Confluence pages, transitions, comments

mcp-atlassian

Ditto

Style guide rules, text search, copy validation

ditto-mcp

Bitbucket

Browse repos, review PRs, search code

bitbucket-mcp

Recraft

AI image generation, vectorization, background removal

recraft-mcp

Qualtrics

Survey creation, response analysis

qualtrics-mcp
